2. 基本功能：

支持6路常开/常闭报警器

支持WiFi和MicroSIM卡双网使用

支持本机布撤防和手机APP布撤防

报警信息立即发送给物业管理机和住户手机
3. 功能使用：

当出现网络故障时，请及时检查网络，否则无法正常报警

布防：撤防状态下单击按键即可布防，布防指示灯亮，语音提示“布防成功”

撤防：布防状态下快速3击按键即可撤防，撤防指示灯亮，语音提示“撤防成功”

报警：发生报警时，红色指示灯闪烁，响起报警声，报警立即发送给物业管理机和安装APP的住户手机

取消报警：报警状态下快速3击按键即可取消报警声，若是防盗类防区报警，则该路报警器会被同时撤防，语音提示“撤防成功”

延时布防：红外等防盗类报警器默认延时布防时间为60秒，便于用户布防后撤离房屋，手机APP可修改延时时间

延时报警：红外等防盗类报警器默认延时报警时间为45秒并有延时报警语音提示，便于用户进入房屋后有时间撤防，防止误报警，手机APP可修改延时时间


WiFi更改:

在撤防状态下长按键3秒进入设置模式

进入“手机APP-设置-配置报警网关WiFi”

选择家庭WiFi并输入WiFi密码

等待配置完成，有语音提示“WiFi设置成功”

单击按键退出设置模式，开始正常使用


Micro SIM卡使用：打开产品外壳，将已开通2G数据功能的移动或联通Micro SIM卡插入卡槽，Micro SIM卡在WiFi未接通情况下自动开启

绿灯闪烁：当网络不通等故障状态下，绿色指示灯闪烁，单击按键可播放故障提示


注：布撤防只对红外等防盗类报警器有效，燃气、紧急等报警器为24小时防区，一直为布防状态，不可撤防。按键布撤防对所有防盗类报警器生效，手机APP可对单个报警器独立布撤防
4. 技术参数：

工作电压：DC 9-18V

额定功率：1.5W

工作温度：-10℃~+55℃

工作湿度：20%～93%

6. 安装：

按要求完成电源及防区接线，此时不上电

将拨片插入拆机槽，上下轻轻拨开产品外壳

插入Mirco SIM卡（可选）

固定螺丝钉（请勿触碰元器件）

给设备上电

用配置工具配置房号及防区信息

进行布撤防操作验证安装是否正确
