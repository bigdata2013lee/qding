1. 为什么无法使用云对讲功能？

云对讲功能必须绑定房屋，请确认APP是否与您的房屋正确绑定，若没绑定或绑定错误，请联系物业绑定
2. 为什么收不到门口机呼叫？

请确认手机已联网

部分手机会清除后台APP，导致手机无法接收通知。请确保在安装本APP时允许所有权限申请

房屋绑定了多个家人时，会按顺序依次呼叫，接听或开锁后不再呼叫

请确认是否开通了免打扰功能

当APP都呼不通时会转呼到该电话，接听后按#键开锁
3. 如何调整接听顺序？

绑定房屋顺序即接听顺序，可进入“云对讲设置-接听顺序”调整接听顺序
4. 如何更换转呼电话号码？

转呼电话号码默认为第一个绑定房屋的手机号，可进入“设置-接听顺序-转接电话”更换号码
5. 为什么门口机呼叫没有视频？

为了节省流量，系统默认处在3G/4G联网时采用语音对讲，可在“设置-3G、4G视频对讲”更改

为了保证对讲效果，当手机网络不畅时会自动切换至语音对讲
