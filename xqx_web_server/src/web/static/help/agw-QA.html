1. 为什么无法使用报警网关功能？

报警网关与房屋绑定，请先确认APP是否与您的房屋绑定，若没绑定房屋，请联系物业绑定

请确认您的房屋是否安装报警网关设备，若已安装却不能使用，请联系物业查看设备配置是否正确
2. 为什么收不到报警通知？

报警通知可以在设置中订阅，系统只推送已订阅的通知，在报警记录中可查看所有通知

部分手机会清除后台APP，导致手机无法接收通知。请确保在安装本APP时允许所有权限申请

物业管理机会在报警时立即收到通知，保障居家安全
3. 家中有多个防盗报警，如何统一布撤防？

报警网关的按键和APP的布撤防按键都是对全部防盗报警器进行布撤防，方便操作
4. 家中有多个防盗报警，如何单个布撤防？

在APP的防区信息里，可进入单个防盗报警器进行布撤防
5. 为什么防盗报警撤防后，还是显示布防状态？

当房屋有多个红外、门磁、窗磁等防盗报警器时，只要有一个处在布防状态，则报警网关和APP上显示为布防
6. 为什么紧急报警、燃气报警、烟感报警不能撤防？

紧急、燃气、烟感属于24小时防区，为了持续保障安全，不可撤防。如有需要请联系物业处理
7. 为什么点击3次报警网关按键不能撤防？

撤防时请快速点击3次，如果间隔较大或次数不对，系统会认为是误操作
8. 如何更换报警网关的SIM卡？

SIM卡最好在首次安装时插入，后期更换需要联系物业将设备断电，取下外壳，按指引方向插入SIM卡，合上外壳，最后上电
9. 什么是延时布防？

红外等防盗类报警器默认延时布防时间为60秒，即布防60秒后开始检测报警，便于用户布防后撤离房屋，手机APP可修改延时时间
10. 什么是延时报警？

红外等防盗类报警器默认延时报警时间为45秒，触发后有延时报警语音提示，倒计时结束没有撤防则立即报警。便于用户进入房屋后有时间撤防，防止误报警，手机APP可修改延时时间
11. 什么是延时报警通知？

当红外等防盗报警器触发时就先向APP发送通知，第一时间掌握人员进出情况
